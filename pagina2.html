Para links internos utilizamos o:

rel="prev" para página anterir.

rel="next" para página posterior.

rel="nofolow" utilizado em links externos para informar ao navegador para não indexar a página, utilizado em propagandas

Embora funcione sem inserir essas informações é importante adiciona-las,
para que o sistema busca entenda como funciona a Hierarquia de páginas do nosso site.
